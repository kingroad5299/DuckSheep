## ✨ 特色

- 🥚 **5 个进化阶段** — 从蛋 → 宝宝 → 成长 → 成熟 → 完全体
- 📊 **4 维状态系统** — 饱腹度、快乐度、体力、清洁度实时变化
- 🎮 **4 种互动方式** — 喂食 🍞、玩耍 ⚽、清洁 🛁、哄睡 😴
- 💾 **自动存档** — 关闭浏览器也不会丢失进度，回来继续养
- 🏆 **12 个成就** — 解锁成就获得满满成就感
- ✨ **粒子特效** — 每次互动都有绚丽特效
- 🔊 **音效反馈** — Web Audio API 生成，无需额外文件
- 🌙 **夜间模式** — 宠物睡觉时自动切换暗色主题
- 📱 **响应式设计** — 手机、平板、桌面全适配
- ⚡ **纯前端** — 零依赖，双击 HTML 就能跑

## 🎮 玩法

| 操作 | 效果 | 消耗 |
|------|------|------|
| 🍞 **喂食** | 饱腹度 +25 | 清洁度 -5 |
| ⚽ **玩耍** | 快乐度 +25 | 体力 -15, 清洁度 -8 |
| 🛁 **清洁** | 清洁度 +30 | 快乐度 +5 |
| 😴 **睡觉** | 体力快速恢复 | 饱腹度/快乐度缓慢下降 |
| 👆 **点击宠物** | 快乐度 +3 | 无 |

> 💡 **提示：** 各项属性会随时间缓慢下降。保持平衡才能让宠物开心！

## 🛠 技术栈

- **HTML5** — 语义化结构
- **CSS3** — 渐变背景 + 动画 + 响应式
- **Vanilla JavaScript** — 零依赖
- **Web Audio API** — 音效生成
- **LocalStorage** — 数据持久化
- **CSS Custom Properties** — 主题切换
