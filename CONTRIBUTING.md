# 🤝 贡献指南

感谢你对 DuckSheep 的关注！欢迎任何形式的贡献。

## 如何贡献

### 🐛 报告 Bug

1. 在 Issues 中搜索是否已有相同问题
2. 如果没有，创建新 Issue，包含：
   - 浏览器版本
   - 复现步骤
   - 预期行为 vs 实际行为
   - 截图（如有）

### 💡 功能建议

1. 在 Issues 中描述你的想法
2. 说明为什么这个功能对项目有价值
3. 如果可以，描述实现思路

### 🔧 提交代码

1. Fork 本仓库
2. 创建特性分支：git checkout -b feature/amazing-feature
3. 提交更改：git commit -m 'feat: add amazing feature'
4. 推送到分支：git push origin feature/amazing-feature
5. 创建 Pull Request

### 提交规范

使用 [Conventional Commits](https://www.conventionalcommits.org/)：

- eat: 新功能
- ix: 修复 Bug
- style: 样式调整
- efactor: 代码重构
- docs: 文档更新
- chore: 构建/工具变更

## 代码风格

- 使用 ES6+ 语法
- 2 空格缩进
- 函数名使用 camelCase
- CSS 类名使用 kebab-case
